This app uses your Google Sheet as a live database. Complete the one-time setup below to get started.
1
Open your Google Sheet → Extensions → Apps Script
2
Paste the Code.gs file contents, replacing any existing code. Then run setupSheets() once.
3
Deploy → New Deployment → Web App. Set Execute as: Me and Who has access: Anyone.
4
Copy the deployment URL and paste it below.
